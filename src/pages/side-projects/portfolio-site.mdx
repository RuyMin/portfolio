## Overview

이력서에 연결할 수 있는 개인 포트폴리오와 프로젝트 기록 공간을 만들기 위해 시작한 프로젝트입니다.

## Why I Built This

10년간 진행한 실무 프로젝트와 기술 경험을 한 곳에 정리하고, 이후 개인 프로젝트를 누적하기 위한 기반이 필요했습니다.

## Features

- 경력 타임라인
- 대표 프로젝트 케이스 스터디
- Side Projects 목록
- GitHub Pages 자동 배포
- 반응형 레이아웃

## Technical Decisions

- 정적 배포와 문서 기반 관리를 위해 Astro를 선택했습니다.
- GitHub Pages 배포를 위해 GitHub Actions를 구성했습니다.
- 프로젝트 사이트 경로 대응을 위해 Astro `base` 설정을 적용했습니다.

## What I Learned

- Astro의 정적 라우팅 구조
- GitHub Pages 프로젝트 사이트 배포 방식
- base path 환경에서 asset/link 경로를 처리하는 방법
