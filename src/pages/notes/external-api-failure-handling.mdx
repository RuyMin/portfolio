## Situation

외부 시스템과 연동할 때 장애는 예외 상황이 아니라 정상적으로 고려해야 하는 설계 조건입니다.

## Checklist

1. 타임아웃 기준을 명확히 둡니다.
2. 재시도 가능한 오류와 불가능한 오류를 구분합니다.
3. 중복 요청이 발생해도 문제가 없도록 idempotency를 고려합니다.
4. 실패 이력을 추적 가능한 형태로 저장합니다.
5. 사용자에게 보여줄 상태와 운영자가 확인할 상태를 분리합니다.
