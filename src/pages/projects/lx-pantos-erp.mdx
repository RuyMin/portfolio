## Overview

- 기간: 2022.07 - 2023.01
- 역할: ERP 업무 화면 및 SAP 연동 기능 개발
- 도메인: 물류/ERP
- 대상 지역: 국내, 멕시코, 러시아
- 기술: Java, Spring Framework, Vue.js, OracleDB, XML Interface, SAP 연동

## Context

LX판토스 ERP 시스템 구축 프로젝트에 참여했습니다. 국내, 멕시코, 러시아 법인 업무에 대응하는 ERP 화면과 인터페이스를 개발했습니다.

주요 업무 영역은 매입계산서/세금계산서 발행 시스템과 직원 출장 및 휴가 관리 화면이었습니다. 
세금계산서 데이터는 SAP 시스템과 연동하여 처리했으며, Java 애플리케이션과 SAP 간 XML 기반 인터페이스를 통해 데이터를 송수신했습니다.

## My Contributions

- 국내, 멕시코, 러시아 업무 요건에 대응하는 ERP 화면 및 백엔드 로직을 개발했습니다.
- 매입계산서 및 세금계산서 발행 기능을 구현했습니다.
- SAP 연동을 위한 XML 요청/응답 데이터 처리 로직을 개발했습니다.
- 직원 출장 및 휴가 관리 화면을 개발했습니다.
- 화면 입력값, 업무 검증, 저장/조회 흐름을 백엔드 API와 연계했습니다.

## Problems

### 국가별 ERP 업무 요건 대응
국내, 멕시코, 러시아 법인별로 계산서/세금계산서 처리 방식과 화면 요구사항이 달라, 공통 구조 안에서 국가별 업무 요건을 반영해야 했습니다.

### SAP 연동 데이터 처리
세금계산서 발행에 필요한 데이터를 SAP와 연동해야 했고, XML 기반 요청/응답 데이터를 Java 애플리케이션에서 안정적으로 처리해야 했습니다.

### 업무 화면의 처리 흐름 정리
매입계산서/세금계산서 발행, 출장/휴가 관리 화면에서 사용자가 반복적으로 수행하는 입력, 조회, 승인/처리 흐름을 화면과 백엔드 로직으로 구현해야 했습니다.

## Technical Points

- 국가별 업무 차이를 공통 화면/로직 안에서 분기 처리
- XML 기반 SAP 인터페이스 요청/응답 처리
- ERP 업무 화면의 입력, 조회, 검증, 저장 흐름 구현
- 계산서/세금계산서 발행 데이터의 정합성 검증

## Impact

- 국내외 법인 업무에 대응하는 ERP 기능 구축에 기여했습니다.
- SAP와 Java 애플리케이션 간 세금계산서 데이터 연동 흐름을 구현했습니다.
- 반복적인 계산서 발행 및 출장/휴가 관리 업무를 ERP 화면에서 처리할 수 있도록 지원했습니다.
