## Overview

- 기간: 2020.12 - 2022.07
- 역할: 백엔드 / 서버 개발 핵심 멤버
- 도메인: 모바일 AI 서비스
- 기술: Java, Spring Framework, REST API, Docker, MSA, ELK, AI 인프라 연동

## Context

리얼네트웍스의 모바일 통화 AI 스팸차단 서비스 KONTEXT의 코어 API와 서버 아키텍처를 담당했습니다. 모바일 수신 통화 데이터를 실시간으로 분석하고, AI 판단 결과를 기반으로 스팸 여부를 안정적으로 필터링하는 것이 핵심 과제였습니다.

AI 인프라 연동 데이터 파이프라인을 백엔드 서버에 내재화하고, 운영 분석을 위한 로그 수집 환경을 구축했습니다.

## My Contributions

- 실시간 통화 분석을 위한 코어 API를 개발했습니다.
- 대용량 트래픽 제어를 고려한 서버 아키텍처를 개선했습니다.
- AI 인프라 연동 데이터 파이프라인을 백엔드 서버에 내재화했습니다.
- 로그 수집과 운영 분석을 위한 ELK 기반 관측 환경을 연계했습니다.

## Problems

### 실시간 대용량 트래픽 처리
모바일 수신 통화마다 AI 분석 요청이 발생하여, 대용량 동시 요청을 안정적으로 처리할 수 있는 서버 구조가 필요했습니다.

### AI 인프라 연동 안정성
AI 분석 인프라와 모바일 서비스 백엔드 간 데이터 흐름에서 지연이나 장애가 발생할 경우 서비스 전체에 영향을 줄 수 있어, 안정적인 파이프라인 설계가 필요했습니다.

### 운영 가시성 확보
실시간 서비스 특성상 장애 감지와 원인 분석을 빠르게 수행할 수 있는 로그/모니터링 환경이 필요했습니다.

## Technical Points

- 실시간 통화 분석 코어 API 설계 및 트래픽 제어
- AI 인프라 연동 데이터 파이프라인 내재화
- Docker 기반 MSA 구조에서의 서비스 분리
- ELK 기반 로그 수집 및 운영 관측 환경 구축

## Impact

- AI 분석 인프라와 모바일 서비스 백엔드 간 데이터 흐름을 안정화했습니다.
- 대용량 요청 처리에 필요한 서버 구조 개선에 기여했습니다.
- 운영 분석 환경을 통해 장애 감지 및 대응 속도를 높였습니다.
