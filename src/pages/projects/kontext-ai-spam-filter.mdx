## Overview

- 기간: 2020.12 - 2022.07
- 역할: 백엔드 / 서버 개발 핵심 멤버
- 기술: Java, Spring Framework, REST API, Docker, MSA, ELK, AI 인프라 연동

## Problem

모바일 수신 통화 데이터를 실시간으로 분석하고, AI 판단 결과를 기반으로 스팸 여부를 안정적으로 필터링해야 했습니다.

## What I Did

- 실시간 통화 분석을 위한 코어 API 개발
- 대용량 트래픽 제어를 고려한 서버 아키텍처 개선
- AI 인프라 연동 데이터 파이프라인을 백엔드 서버에 내재화
- 로그 수집과 운영 분석을 위한 ELK 기반 관측 환경 연계

## Impact

- AI 분석 인프라와 모바일 서비스 백엔드 간 데이터 흐름을 안정화했습니다.
- 대용량 요청 처리에 필요한 서버 구조 개선에 기여했습니다.
