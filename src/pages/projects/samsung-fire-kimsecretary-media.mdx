## Context

삼성화재 전사 영업 지원 플랫폼 김비서 앱에 모바일 영상 녹화 기능을 추가하는 프로젝트에 참여 예정입니다. 비대면 영업 및 보안 프로세스 강화를 위해 앱 내 멀티미디어 인프라를 구축하고, 백엔드 연동을 담당합니다.

## My Contributions

- 비대면 영업 및 보안 프로세스 강화를 위한 모바일 영상 녹화 인프라를 구축합니다.
- 앱 내 멀티미디어 데이터 스트리밍 REST API를 설계합니다.

## Problems

### 모바일 영상 데이터 처리
모바일 환경에서 촬영된 영상 데이터를 안정적으로 서버에 전송하고, 인코딩 서버와 연동하여 처리해야 합니다.

### 스트리밍 API 설계
대용량 멀티미디어 데이터를 효율적으로 전송하기 위한 스트리밍 API 구조를 설계해야 합니다.

### 보안/권한 정책
영업 관련 영상 데이터의 보안 요구사항과 접근 권한 정책을 반영해야 합니다.
