- 기술: Java, Spring Boot, Spring Framework, Vue.js, OracleDB, REST API, MSA

## Context

삼성화재 장기보험 코어 시스템 고도화 프로젝트에 참여했습니다. 여러 하위 프로젝트를 수행했으며, 그 중 장기설계 고도화가 메인 업무였습니다.

장기설계 고도화 과정에서 본사에서 개발한 AI 모델을 설계 프로세스에 도입하는 작업을 진행했습니다. AI 자체를 개발한 것이 아니라, 본사 AI와 코어 시스템을 연동하는 인프라를 구축했습니다. 다만 폐쇄망 환경이었기 때문에 연동 과정에서 네트워크 제약과 보안 정책을 고려한 설계가 필요했습니다.

원장 데이터 연계가 필요했기 때문에, SAP 개발자와 긴밀히 소통하며 연계 구조를 설계하고 개발을 진행했습니다.

## My Contributions

- 장기설계 고도화 프로젝트의 코어 백엔드 개발 및 파트 리딩을 수행했습니다.
- 본사 AI 모델과 코어 시스템 간 연동 인프라를 폐쇄망 환경에서 구축했습니다.
- SAP 개발자와 협업하여 원장 데이터 연계 구조를 설계하고 구현했습니다.
- 다수 설계사가 동일 설계를 동시 수정할 때 발생하는 동시성 병목을 분석하고 해결했습니다.

## Problems

### 폐쇄망 환경에서의 AI 연동
본사 AI 모델을 장기설계 프로세스에 도입해야 했지만, 폐쇄망 환경의 네트워크 제약과 보안 정책으로 인해 일반적인 API 연동 방식을 그대로 적용할 수 없었습니다.

### 원장 데이터 연계
장기설계에 필요한 원장 데이터 연계를 위해 SAP 개발자와의 협업 프로세스를 정립하고, 시스템 간 데이터 흐름을 설계해야 했습니다.

### 설계사 간 동시 수정 병목
다수의 설계사가 동일한 설계 건을 동시에 수정할 때 데이터 충돌과 병목이 발생할 수 있어, 동시성 제어 방안을 설계해야 했습니다.

## Technical Points

- 폐쇄망 환경에서의 AI 모델 연동 인프라 설계
- 원장 데이터 연계 구조 설계 및 SAP 개발자와의 크로스팀 협업
- 다수 설계사 동시 수정 시 동시성 제어 설계

## Impact

- 장기설계 프로세스에 AI를 도입하여 업무 효율을 개선했습니다.
- 폐쇄망 제약 하에서 안정적인 AI 연동 인프라를 구축했습니다.
- 오픈 전 동시성 이슈를 해결하고 시스템 안정성을 확보했습니다.
